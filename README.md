## 📦 Installation
Make sure you have Python installed, then install Pygame:  

```bash
pip install pygame
```

---
## ▶️ How to Play
Run the game with:  

```bash
python main.py
```

🎮 Controls:
- Arrow keys → Move the snake 🏹
- Eat food → Grow longer 🍏
- Avoid collisions → Don't crash into yourself!
